# React Image Editor

A powerful and intuitive web-based image editor built with React and Tailwind CSS.

## Features

- **Basic Tools**: Crop, Rotate, Resize, Grayscale.
- **Advanced Tools**: 
  - **Cutout**: Remove backgrounds manually by color or remove black background (Unmultiply).
  - **Collage**: Create photo grids and layouts.
  - **Adjust**: Fine-tune Hue, Saturation, and Brightness.
  - **Converter**: Convert images to PNG, JPG, or WEBP with quality control.
  - **Format Support**: Supports standard images plus TGA files.
- **Theming**: Customizable UI colors.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deployment (How to share with others)

The easiest way to deploy this app is using **Vercel** or **Netlify**.

1. Push this code to a Git repository (GitHub, GitLab, etc.).
2. Import your project into your preferred hosting provider.
3. Click **Deploy**.
